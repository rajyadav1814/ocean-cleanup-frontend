# Ocean Cleanup Frontend

This is the React frontend for the Ocean Cleanup tracking platform. It provides specialized dashboards for contributors, verifiers, and the general public to track and verify ocean cleanup efforts worldwide.

## Features

- **Contributor Workspace**: Allows users to log cleanup activities, upload evidence photos, and track their impact credits.
- **Verifier Workspace**: A dedicated queue for verifiers to review submitted activities and approve or reject them.
- **Public Dashboard**: Real-time tracking of global cleanup efforts, organization leaderboards, and an interactive impact map.
- **Premium Design**: Built with a custom glassmorphism aesthetic and responsive layouts.

## Tech Stack

- React
- React Router DOM
- Vite
- Custom CSS (Oceanic Premium Dark Theme)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to the local development URL (typically `http://localhost:5173` or similar).

## Environment Variables

Ensure that the backend URL is properly configured in your environment or API service layer so that requests are routed to the backend service.
